# Flask Calculator

A simple web-based calculator developed using Python and Flask. This application performs basic arithmetic operations through an easy-to-use interface and demonstrates the fundamentals of Flask web development.

## Live Demo

Live Application:
https://flask-calculator-wqyj.onrender.com

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Percentage Calculation
- Input Validation
- Division by Zero Error Handling
- Responsive User Interface using Bootstrap
- Clear Button
- Preserves User Input After Calculation

## Installation

### Clone the Repository

```bash
git clone https://github.com/niyati-yele/flask-calculator.git
```

### Move to the Project Directory

```bash
cd flask-calculator
```

### Create a Virtual Environment

```bash
python -m venv venv
```

### Activate the Virtual Environment

**Windows**

```bash
venv\Scripts\activate
```

**Linux/macOS**

```bash
source venv/bin/activate
```

### Install Required Packages

```bash
pip install -r requirements.txt
```

### Run the Application

```bash
python app.py
```

Open your browser and visit:

```
http://127.0.0.1:5000/
```
